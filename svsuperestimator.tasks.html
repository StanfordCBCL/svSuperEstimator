class svsuperestimator.tasks.task.Task(project, config, prefix='', parent_folder=None, log_config=True)[source]#

Bases: ABC

Base class for svSuper Estimator task.

A task is a set of actions to be performed on a SimVascular project. Each task defines 3 routines. A core_run routine in which the main calculation of the task is performed, a post_run routine where the data is postprocessed and a generate_report routine, where the postprocessed results are visualized.
