update_simparams(abs_tol=None, max_nliter=None, num_cycles=None, steady_initial=None, mean_only=None, output_interval=None, last_cycle_only=None, variable_based=None)[source]#

Update the simulation parameters.

Parameters:

  • abs_tol (Optional[float]) – Absolute tolerance for simulation.

  • max_nliter (Optional[int]) – Maximum number of non-linear iterations per time step in time integration.

  • num_cycles (Optional[float]) – Number of cardiac cycles to simulate.

  • steady_initial (Optional[bool]) – Solve steady solution first and use as intitial condition.

  • mean_only (Optional[bool]) – Return only mean values over time steps.

  • output_interval (Optional[bool]) – Interval for writing a timestep to the output.

  • last_cycle_only (Optional[bool]) – Output only last cycle.

  • variable_based (Optional[bool]) – Node based output.
